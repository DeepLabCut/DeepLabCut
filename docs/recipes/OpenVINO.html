Intel OpenVINO backend#

DeepLabCut provides an option to run deep learning model with OpenVINO backend. To enable OpenVINO in your pipeline, use use_openvino flag of analyze_videos method with one of string values indicating device:

  • ‚ÄúCPU‚Äù - Use CPU. This is a default value.

  • ‚ÄúGPU‚Äù - Use iGPU (requires OpenCL to be installed). First launch might take some time for kernels initialization.

  • ‚ÄúMULTI:CPU,GPU‚Äù - Use CPU and GPU simultaneously. In most cases this option provides the best efficiency.

def analyze_videos(
    ...
    use_openvino="MULTI:CPU,GPU",
)

OpenVINO is an optional dependency. You can install it with DeepLabCut by the following command:

pip install deeplabcut[openvino]
